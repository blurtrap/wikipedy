# 📚 Моя Википедия

Современная вики-платформа с темной темой, стеклянным дизайном и сохранением статей в файлы.

## ✨ Особенности

- 🌙 **Темная тема по умолчанию** - приятная для глаз
- 🪟 **Стеклянный дизайн (Glassmorphism)** - современный и красивый интерфейс
- 📁 **Сохранение в файлы** - все статьи сохраняются в папку `article/`
- ✍️ **Упрощенная публикация** - просто создайте и сохраните
- 🔍 **Умный поиск** - поиск в реальном времени
- 💾 **Автосохранение** - ваши изменения сохраняются автоматически
- 🗑️ **Управление статьями** - редактирование и удаление
- 📱 **Адаптивный дизайн** - работает на всех устройствах

## 🚀 Быстрый старт

### Установка

1. Убедитесь, что у вас установлен Node.js (версия 14 или выше)

2. Установите зависимости (если нужно):
```bash
npm install
```

3. Запустите сервер:
```bash
npm start
```

4. Откройте браузер и перейдите на:
```
http://localhost:3000
```

## 📖 Использование

### Создание статьи

1. Нажмите кнопку **"➕ Создать новую статью"**
2. Введите название и содержание
3. Нажмите **"💾 Сохранить"** или просто продолжайте писать - автосохранение сработает через 2 секунды

### Редактирование статьи

1. Откройте статью
2. Нажмите **"✏️ Редактировать"**
3. Внесите изменения
4. Сохраните

### Поиск

- Используйте поисковую строку для быстрого поиска статей
- Поиск работает в реальном времени при вводе текста

### Форматирование текста

- `# Заголовок` - заголовок первого уровня
- `## Подзаголовок` - заголовок второго уровня
- `**жирный текст**` - жирный текст
- `*курсив*` - курсив
- `[[Название статьи]]` - ссылка на другую статью
- Пустая строка - новый абзац

```
├── index.html          # Главная страница
├── article.html        # Страница просмотра статьи
├── edit.html          # Страница редактирования
```

## 🎨 Особенности дизайна

- **Темная тема** - снижает нагрузку на глаза
- **Glassmorphism** - эффект матового стекла для элементов
- **Градиенты** - красивые цветовые переходы
- **Анимации** - плавные переходы и эффекты
- **Карточки статей** - удобное отображение списка статей

## 🔧 API Endpoints

- `GET /api/articles` - получить все статьи
- `GET /api/articles/:title` - получить статью по названию
- `POST /api/articles` - создать/обновить статью (**требуется admin-пароль**)
- `DELETE /api/articles/:title` - удалить статью (**требуется admin-пароль**)

## 🔐 Пароль администратора

- На сервере есть переменная `ADMIN_PASSWORD` (по умолчанию `admin1234`).
- Рекомендуется запускать сервер так:

```bash
set ADMIN_PASSWORD=мой_секретный_пароль && npm start
```

- При первом сохранении/удалении статья браузер спросит пароль и будет отправлять его в заголовке `X-Admin-Password`.
- Без правильного пароля никто (даже с другого IP) не сможет изменить или удалить статьи.

## 💡 Полезные функции

- **Автосохранение** - изменения сохраняются автоматически через 2 секунды после остановки ввода
- **Поиск в реальном времени** - результаты обновляются при вводе
- **Удаление статей** - кнопка удаления прямо в карточке статьи
- **Информация о дате** - показывает когда была создана статья
- **Fallback на localStorage** - если сервер недоступен, используется локальное хранилище браузера

## 📝 Формат статей

Статьи сохраняются в формате JSON:

```json
{
  "title": "Название статьи",
  "content": "Содержание статьи...",
  "author": "user",
  "createdAt": "2024-01-01T00:00:00.000Z",
  "updatedAt": "2024-01-01T00:00:00.000Z"
}
```
