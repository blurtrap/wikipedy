 Моя Википедия 📚

Простая вики-платформа на чистом JavaScript, HTML и CSS. Все статьи хранятся в вашем браузере!

## 🌟 Возможности

- ✍️ Создание новых статей
- 📝 Редактирование существующих
- 🔍 Поиск статей
- 🔗 Вики-ссылки [[в двойных скобках]]
- 💾 Сохранение в localStorage браузера
- 📱 Адаптивный дизайн
- 🚀 Работает на GitHub Pages

## 🎮 Демо
